# MaLeMa - (MAX LEO MAT)

## Configuration

1. `cd frontend`
2. `npm install`

### Project Start in local

```
npm run start
```

### Project Build (for deploy)

```
npm run build
```

## Dependencies

- **NodeJs** `8.9.x`
- **npm** `5.6.x`
- **Bootstrap** `4.x`.
- **Eslint** `^4.18.2` and **Eslint Plugin React** `^7.7.0`.
